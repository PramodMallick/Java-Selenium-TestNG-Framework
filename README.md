# Selenium WebDriver TestNG Framework
Automating functional and end to end UI tests for www.oculus.com website. This project uses the page object model (POM) design. 

## Stack & Libraries
- Java
- Selenium WebDriver
- TestNG 
- Extent Reports and logging

# Prerequisites
- IDE (Eclipse)
- JDK (version 7 or 8)
- Maven 
- Chrome version 80+
